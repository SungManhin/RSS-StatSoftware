数据结构：矩阵

  • 将向量摞起来

  • 矩阵是一个二维数组，外表类似于 excel 中的表格

  • 每个元素都必须拥有相同的数据类型

  • 是在各种数学模拟运算中用到最多的数据结构

rARPACK:大规模矩阵运算

  • 这个R包可以提高大规模矩阵的运算效率(例如：矩阵求逆，特征值、奇异值求解等)。

  • 仅对矩阵计算一部分有代表性的特征值（奇异值）来近似分解，类似于主成分分析中只选取前几个主成分来概括原始信息的思想。

  • 对比实例

#打开这个包
library(rARPACK)
#构造一个1000维的大型矩阵
T=matrix(1:1000000,1000,1000)
#正常分解与快速分解的对比，此处以选择前5个特征（奇异）值为例
system.time(svd(T))
##    user  system elapsed 
##    2.44    0.00    2.44
system.time(svds(T,5)) 
##    user  system elapsed 
##    0.07    0.00    0.07
system.time(eigen(T))
##    user  system elapsed 
##    4.19    0.00    4.19
system.time(eigs(T,5))
##    user  system elapsed 
##     0.2     0.0     0.2

稀疏矩阵

  • 定义：矩阵所包含的元素中，数值为 0 的元素远远多于非 0 元素。

  • 示例：把微博上用户之间的关注数据抽象成邻接矩阵 A

  • 稀疏矩阵的生成
library(Matrix)

m1 <- matrix(vector, nrow = 3, ncol = 4);m1

稀疏矩阵：优势

  • 内存优势

m1 <- matrix(0, nrow = n, ncol = n)
m2 <- Matrix(0, nrow = n ,ncol = n, sparse = TRUE)
object.size(m1)
## 1016 bytes
object.size(m2)
## 1768 bytes
system.time(matrix(0, nrow = n, ncol = n))
##    user  system elapsed 
##       0       0       0
system.time(Matrix(0, nrow = n ,ncol = n, sparse = TRUE))
##    user  system elapsed 
##       0       0       0
#行列数分别为100时，稀疏矩阵在内存优势已经很大了，但生成时间还略有落后
